# Queer Boots
This is the repo for the (currently in development) queer boots website

## Getting up and running
```
git clone https://github.com/queerboots/queerboots.git
cd queerboots
npm install
```
At this point, copy the `.env.example` file to `.env` and fill in the values as needed.

## Running
### Development
```
npm run start
```

### Production
```
npm run build
npm run serve
```
